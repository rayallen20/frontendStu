# 05-盒子模型边框

## 1. 盒子模型的作用

- 作用: 布局网页,摆放盒子和内容

## 2. 盒子模型的组成

- 内容区域: `width`/`height`
  - 调试工具中显示为蓝色 
- 内边距: `padding`(出现在内容与盒子边缘之间)
  - 通常内容和边框之间的间距使用`padding`实现,而非`text-indent`
  - 调试工具中显示未绿色
- 边框线: `border`
- 外边距: `margin`(出现在盒子外面)
  - 调试工具中显示为黄色

## 3. 边框的属性

- 属性名: `border`
- 属性值: `边框线粗细 线条样式 颜色`(**不区分顺序**)

## 4. 常用的线条样式

|   属性值    | 线条样式 |
|:--------:|:----:|
| `solid`  |  实线  |
| `dashed` |  虚线  |
| `dotted` |  点线  |
