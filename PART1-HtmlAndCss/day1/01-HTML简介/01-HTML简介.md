# 01-HTML简介

## 1. HTML定义

HTML:超文本标记语言,HyperText Markup Language

- 超文本是什么?

超文本特指链接

链接:通过链接可以把多个网页链接到一起

- 标记是什么?

标记:也叫标签,带尖括号的文本

## 2. 标签语法

- 标签成对出现,中间包裹内容,分为开始标签和结束标签
- `<>`里面放英文字母(标签名),一般都是小写
- 结束标签比开始标签多一个`/`
- 扩展
  - 单标签:只有开始标签,没有结束标签
  - 双标签:成对出现的标签,有开始标签和结束标签

# 3. 总结

1. 保存HTML标签的文件扩展名是什么?

    `.html`
2. HTML标签名要放在什么括号中?

    `<>`
3. 结束标签比开始标签多了什么?

    `/`
4. 标签包裹的内容放在什么位置?

    开始标签和结束标签之间

# 4. 初体验

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
<strong>加粗文字</strong> 不加粗文字
</body>
</html>
```
