# 04-某个弹性条目侧轴对齐方式

## 1. 属性名与属性值

- 属性名: `align-self`
- 属性值:

|      属性值      |             效果             |
|:-------------:|:--------------------------:|
|    `auto`     | 默认值,继承父元素的`align-items`属性值 |
|   `stretch`   |  拉伸填充弹性容器(填充到与同行的弹性条目一样高)  |
| `flex-start`  |            侧轴起点            |
|  `flex-end`   |            侧轴终点            |
| `flex-center` |         相对父元素居中对齐          |

## 2. 注意

- 该属性添加在**弹性条目**上
