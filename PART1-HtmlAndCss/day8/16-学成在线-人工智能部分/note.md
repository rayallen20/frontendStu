# 16-学成在线-人工智能部分

## 1. 结构与样式

- 1200 * 316px
- 下外边距32px
- 头部+身体

## 1.1 头部

- 1200 * 30
- 下外边距15px
- h3 + ul + a
- 注意ul里用a的外边距撑开li的宽度

## 1.2 身体

- 1200 * 271
- li: 228 * 271
- img + h4 + p
- 图片下外边距: 20px
- h4: 左右外边距20px 高度50(让底下的p对齐)
- p: 左右外边距20px

## 2. 讲解

- 这块他就完全复用的精品课程模块的样式
- 注意:复制之后,ul的宽度不对了,给ul加一个类名单独调一下宽度就可以了
  - 出现这个问题是因为之前是多行的,ul到达父元素宽度也放不下那么多弹性条目,所以ul的宽度和父元素一致
  - 而这里是单行弹性条目,ul的宽度和弹性条目宽度之和默认是一样的,所以需要调整ul的宽度
