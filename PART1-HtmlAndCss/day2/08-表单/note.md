# 08-表单

## 1. 作用与使用场景

作用: 收集用户信息

使用场景:

- 登录页面
- 注册页面
- 搜索区域

## 2. `input`标签的基本使用

| `type`属性值  |                       说明                       |
|:----------:|:----------------------------------------------:|
|   `text`   |                  文本框,用于输入单行文本                  |
| `password` |                      密码框                       |
|  `radio`   | 单选框.需要多个`radio`有相同的`name`属性,才能在这些`radio`之间实现单选 |
| `checkbox` |                      复选框                       |
|   `file`   |                      上传文件                      |
