# 02-小兔鲜-footer部分

## 1. 上盒子

- 高度: 178px
- 上内边距: 60px
- 下外边距: 60px
- 有版心
- li左外边距: 90px
- 精灵盒子: 58 * 58
- 精灵盒子右外边距: 20px
- 字号: 28px

## 2. 中盒子

- 下外边距: 101px
- dl左外边距: 84px
- dt高度: 25(字体大小) + 30(下外边距) = 55
- dd高度: 22(字体大小) + 11(下外边距) = 33
- 图片: 120 * 120
- 图片下外边距: 10px
- 这里图片和文字装在一个div里,这个div给宽度,限制图片,不给高度,让内容撑开即可

## 3. 底盒子

- ul和span要求水平垂直居中,所以这里要修改主轴方向
  - 实际上没有垂直居中,因为盒子的高度完全由内容撑开的
- 里边的ul和span水平居中实际上是侧轴对齐方式: `align-items: center;`
- ul和span之间的距离用下外边距实现,不能用行高,因为行高是上下的,这里只要下外边距
- a的左右内边距: 5px
- a有右内边框,最后一个没有右边框
