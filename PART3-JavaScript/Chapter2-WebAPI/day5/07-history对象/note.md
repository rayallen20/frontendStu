# 07-history对象

- `window.history`: 该属性是一个对象,该对象用于管理历史记录,可以通过该对象在浏览器历史记录中前进或后退
- 使用场景:
  - 一般在OA系统中用的多,比如在OA系统中,点击一个链接,跳转到另一个页面,然后点击返回按钮,返回到上一个页面
- 方法:
  - `history.back()`: 后退
  - `history.forward()`: 前进
  - `history.go()`: 前进或后退
    - 参数:
      - `0`: 刷新当前页面
      - 正数: 前进
        - 例: `1`: 前进1个页面
      - 负数: 后退
        - 例: `-1`: 后退1个页面
