# 08-案例-随机轮播图

## 1. 需求

- 刷新页面时,页面中的轮播图会显示不同图片以及样式
- 页面刷新时,随机显示图片/文字/背景色

## 2. 样式

- container: 图片盒子 + 下方内容盒子
- 图片盒子: ul > li
  - ul宽度: li个数 * 100%
- 内容盒子: 上下2个div
  - 上div: flex布局; h4 + div(放左右箭头按钮的)
    - 左右箭头盒子: flex布局; a * 2 (a设置为块级元素)
  - 下div: ul > li
    - li里边是a(`href=javascirpt:;`)

## 3. 功能

- 生成对象数组
  - 图片url
  - 标题文字
  - 背景色
- 随机抽取一个索引
  - ul向左移动`索引 / 数组长度 * 100%`
  - 标题内容改变
  - 背景色改变
