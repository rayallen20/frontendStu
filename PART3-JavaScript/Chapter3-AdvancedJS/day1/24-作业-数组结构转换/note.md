# 24-作业-数组结构转换

## 1. 需求

原数组:

```javascript
const input = [
    '生煎',
    '灌汤包',
    '水饺',
    '汉堡',
]
```

请把数组转换为以下格式：

```javascript
const output = [
  {
    label: '生煎',
    value: '生煎',
  },
  {
    label: '灌汤包',
    value: '灌汤包',
  },
  {
    label: '水饺',
    value: '水饺',
  },
  {
    label: '汉堡',
    value: '汉堡',
  },
]
```
