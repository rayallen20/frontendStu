# 07-案例-转换时间

## 1. 需求

- 用户输入秒数,可以自动转换为时分秒

## 2. 思路

### 2.1 布局

- 图片是背景图
- div(背景图) > div(h4 + div*3)
  - 放h4 + div的div,用于调整垂直位置
  - 里边的3个div用flex布局即可

### 2.2 功能

#### 2.2.1 小时

- 输入的数字和3600相除后取整,即为表示小时的数字
- 输入的数字和3600取余,即为表示分钟的秒数

#### 2.2.2 分钟

- 表示分钟的秒数和60相除后取整,即为表示分钟的数字

#### 2.2.3 秒

- 表示分钟的秒数和60取余,即为表示秒的数字
