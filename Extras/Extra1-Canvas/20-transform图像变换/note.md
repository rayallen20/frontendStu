# 20-transform图像变换

- `transform(a, b, c, d, e, f)`: 若一个点的原始坐标为`(x, y)`,则变换后的坐标为:
  - `x = ax + cy + e`
  - `y = bx + dy + f`
- 即:
  - `e`和`f`控制上下文的水平和垂直位移
    - 注意控制的目标是上下文,这一点和位移/旋转缩放是一样的,其目标不是某个具体的点,而是整个上下文(也就是整张画布)
  - 当`b`和`c`为0时:
    - `a`和`d`分别表示上下文的水平和垂直缩放
  - 当`a`和`d`为1时:
    - `b`和`c`分别表示上下文的水平和垂直倾斜

- 换个方式理解:
  - `a`: 水平方向上的缩放
  - `b`: 垂直方向上的倾斜偏移
  - `c`: 水平方向上的倾斜偏移
  - `d`: 垂直方向上的缩放
  - `e`: 水平位移
  - `f`: 垂直位移

变换矩阵的描述:

```
[ 
  a c e
  b d f
  0 0 1
]
```
