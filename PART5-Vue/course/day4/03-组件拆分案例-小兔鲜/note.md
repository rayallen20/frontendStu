# 03-组件拆分案例-小兔鲜

- 全局样式是在`main.js`中引入的

## 1. 拆分局部组件

- 组件拆分,拆分为7个组件,包括其样式也要拆分开:
  - 快捷链接
  - 头部导航
  - 轮播区域
  - 新鲜好物
  - 热门品牌
  - 最新专题
  - 版权底部

## 2. 拆分全局组件


- 将`GoodsArea`组件中的`<li>`作为全局组件`GoodItem`拆分出来
- 在传递数据时,Webpack环境,默认`@`表示`src/`目录
- 模板中的`require()`函数也是因为Webpack的支持,模拟了Node.js的`require()`函数(浏览器默认是不支持`require()`函数的)
