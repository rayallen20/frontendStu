# 01-案例-购物车

## 1. 需求

- 渲染书籍列表
- 显示总价
- `+`/`-` 按钮修改数量
- 移除按钮删除商品
- 当全部商品被移出,显示`购物车为空`

## 2. 实现

- 总价: 应该是个计算属性
  - 只有getter,没有setter,因为总价是由商品的数量和价格计算出来的,没有能直接修改总价的操作
- 加减按钮: 直接在`methods`中定义方法即可,因为修改了`books`,所有**依赖**该响应式数据的地方都会重新渲染
  - 这里也包括计算属性

## 3. 参考答案

- 忘了`v-if`能和`template`标签联用了(说白了是忘了`template`标签里还能套`template`标签了)
