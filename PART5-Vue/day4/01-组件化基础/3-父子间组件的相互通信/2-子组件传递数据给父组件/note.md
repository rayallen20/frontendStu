# 2-子组件传递数据给父组件

使用场景:

- 子组件若想修改Props传递过来的数据,则需要通过事件的方式,通知父组件进行修改
- 或者有一些属于子组件自身的数据,想要传递给父组件,也需要通过事件的方式进行传递

需求:

- `App.vue`根组件中:
  - 显示当前计数
  - 显示`CounterOperation.vue`子组件
- `CounterOperation.vue`子组件:
  - 点击`+1`
  - 点击`-1`
  - 点击`+n`
  - 当上述点击事件发生时,触发事件,通知父组件更新当前计数
